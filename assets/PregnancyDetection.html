Pregnancy Detection with Machine Learning¶

Chao Cheng   |   chaozcheng.com   |   April 2018

Introduction¶

This notebook contains the results of my Pregnancy Detection project, which is based on the traditional Chinese medicinal practice of observing pregnancies through wrist palpations. Using data from women at a women and children's hospital, two different approaches are used to construct a predictive model, the first being a classical machine learning pipeline, and the second being a deep learning technique known as Recurrent Neural Networks (RNNs).

I began this project during the summer of 2017.

Part One: Classical Machine Learning¶

In this first section, I use some common machine learning algorithms utilizing 10-fold cross validation in order to obtain a general model, in addition to taking a closer look at the data.

# plot confusion matrix
                    from sklearn.metrics import confusion_matrix
                    import itertools

                    def plot_confusion_matrix(cm, classes,
                    normalize=False,
                    title='Confusion matrix',
                    cmap=plt.cm.Oranges):
                    """
                        This function prints and plots the confusion matrix.
                        Normalization can be applied by setting `normalize=True`.
                        """
                    if normalize:
                    cm = cm.astype('float') / cm.sum(axis=1)[:, np.newaxis]
                    print("Normalized confusion matrix")
                    else:
                    print('Confusion matrix, without normalization')

                    print(cm)

                    plt.imshow(cm, interpolation='nearest', cmap=cmap)
                    plt.title(title)
                    plt.colorbar()
                    tick_marks = np.arange(len(classes))
                    modify_class = ["Yes","No"]
                    plt.xticks(tick_marks, modify_class, rotation=30)
                    plt.yticks(tick_marks, modify_class)

                    thresh = cm.max() / 2.
                    for i, j in itertools.product(range(cm.shape[0]), range(cm.shape[1])):
                    plt.text(j, i, round(cm[i, j], 2),
                    horizontalalignment="center",
                    color="white" if cm[i, j] > thresh else "black")

                    plt.tight_layout()
                    plt.ylabel('True label')
                    plt.xlabel('Predicted label')

                    # Compute confusion matrix
                    class_names = [1, 0]
                    cnf_matrix = confusion_matrix(y_test, y_pred, labels=class_names)
                    np.set_printoptions(precision=2)

                    # Plot non-normalized confusion matrix
                    plt.figure()
                    plt.grid(b='off')
                    plot_confusion_matrix(cnf_matrix, classes=class_names,
                    title='Confusion Matrix')

                    # Plot normalized confusion matrix
                    plt.figure()
                    plot_confusion_matrix(cnf_matrix, classes=class_names, normalize=True,
                    title='Normalized Confusion Matrix')
                    plt.grid(b='off')
                    plt.show()
